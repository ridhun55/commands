# Install Angular CLI Globally
```shell
npm install -g @angular/cli
```
# Angular Version
```shell
ng version
```
# Angular Help
```shell
ng help
```
# Create Angular Project
```shell
ng new my-project
```
Run Angular Project
```shell
cd my-project
```
```shell
ng serve
```
